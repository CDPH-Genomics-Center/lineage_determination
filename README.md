# lineage_determination
TB lineage determination

```
usage: get_lineage [-h] --vcf VCF --lineages LINEAGES --samplename SAMPLENAME --report REPORT [--verbose]

get lineage

optional arguments:
  -h, --help                              show this help message and exit
  --vcf VCF, -v VCF                       vcf file
  --lineages LINEAGES, -l LINEAGES        lineage information
  --samplename SAMPLENAME, -s SAMPLENAME  sample name
  --report REPORT, -r REPORT              tsv output file
  --verbose                               turn on debugging output

```
